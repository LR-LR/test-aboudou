# Equad's tests for El-haire ABOUDOU

This repo contains Cypress tests for modify password in Equad's test website, developed by El-haire ABOUDOU

## Installation

```
npm install
```

## Run tests in headless mode

```
npm run cy:run
```

## Run tests in head

```
npm run cy:run:head
```

## Open Cypress

```
npm run cy:open
```
